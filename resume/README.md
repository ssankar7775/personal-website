# Resume variants

This directory is organized into separate resume variants:

- `archive/` contains archived exports, including `Santhosh Sankar Resume Aerospace.pdf`
- `Avionics-Embedded-Systems/` contains the current avionics / embedded systems resume
- `Mechanical-Avionics/` contains the starter version for a hybrid mechanical + avionics resume
- `Mechanical/` contains the starter version for a mechanical-focused resume

Each active resume folder can keep its own `resume.tex`, generated PDF, and LaTeX build artifacts.
